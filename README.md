# Touheed Shah - Portfolio Website

A modern, responsive 3D animated portfolio website for Touheed Shah, an aspiring AI Engineer and Full Stack Developer specializing in Robotics.

## Tech Stack

- **Framework:** Next.js 16 with App Router
- **Styling:** Tailwind CSS
- **Animations:** Framer Motion
- **3D Graphics:** Three.js with @react-three/fiber and @react-three/drei
- **Language:** TypeScript

## Features

- ✅ Fully responsive design
- ✅ Fixed navigation with smooth scrolling
- ✅ 3D animated hero section with particles and floating objects
- ✅ Modern UI with dark theme
- ✅ Contact form
- ✅ Mobile hamburger menu
- ✅ Framer Motion animations throughout
- ✅ Clean component-based architecture

## Sections

1. **Hero** - Full-screen intro with 3D background
2. **About** - Personal background and skills
3. **Projects** - Showcase of work (sample projects included)
4. **Experience** - Education and current studies
5. **Contact** - Contact form and social links
6. **Footer** - Simple footer with links

## Getting Started

1. Install dependencies:
```bash
npm install
```

2. Run the development server:
```bash
npm run dev
```

3. Open [http://localhost:3000](http://localhost:3000) in your browser.

## Build for Production

```bash
npm run build
npm start
```

## Deployment

This project can be easily deployed to Vercel, Netlify, or any other platform supporting Next.js.

## Customization

- Update personal information in the components
- Replace placeholder project images with actual screenshots
- Add real project links and descriptions
- Customize colors and animations as needed
